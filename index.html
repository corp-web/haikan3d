視点（カメラ）操作

  • 何もない所からドラッグ＝回転（床下まで回り込めます）
  • ホイールを回す＝拡大／縮小、ホイール押し込みドラッグ＝画面移動、右ドラッグ＝画面移動
  • 右上のビューキューブの面をクリック＝正面・側面など真正面から見る（平行投影）。平行投影中は脇の円弧矢印で画面を90°回転
  • ホーム（⌂）＝最初の視点へ戻る

タッチ操作（iPad / iPhone）

  • 1本指ドラッグ＝回転／2本指ドラッグ＝画面移動／ピンチ＝拡大縮小／タップ＝選択
  • 左下のコントローラーでマウス専用の操作を代替：＋／−＝拡大縮小、向き＝向きを送る（右クリック相当）、回転（Shift＋右クリック相当・旧ひねり）、取消（Esc相当）
  • 配置済みの部品や線を選択中は向き／回転ボタンを長押し（0.35秒）＝角度スピナー（右クリック長押し相当）。▲▼で0.5°刻み（長押しで連続）、3D画面タップで確定、取消ボタンで取消
  • 細かい採寸作業はBluetoothマウスを接続するとPCと同じく右クリック・カーソル吸着が使えます

部品を置く（パレット）

  • 左のパレットで種別を選び、タイルをクリックすると部品がカーソルに付いてきます → 3D空間をクリックして仮置き（起点が近くの機点へスナップ）
  • 仮置き後、プロパティパネルのEL欄に数値を入れEnterで確定
  • 最初の1個は原点(0,0,0)が基準点になります
  • フランジ＝タイプ→呼び径→クラス→フェイス（＋Sch）／継手＝タイプBW・呼び径・Sch／パイプ＝呼び径・Sch・長さ
  • ガスケット＝呼び径・クラス・厚み（既定3mm・任意入力）。外径はそのクラスのRF座面径、内径は管外径。フランジのフェイス間に挟んで使います

部品を動かす・向きを変える

  • 移動（部品も線分・寸法線も）はリボン「編集」の✥移動コマンドで行います。「移動を押す→対象をタップ→ドラッグ」でも「対象をタップ→移動を押す→ドラッグ」でもOK。1回動かすとコマンドは自動で終了し、取消はEscか再押下。コマンド外ではドラッグしても動かないので、誤って動かす心配がありません（選択・起点選択・パイプ端や線端の伸縮・寸法の逃げ調整・円の半径変更・EL入力はいつでも可能。作図や部品配置を始めると移動コマンドは自動解除）
  • 部品をドラッグ＝45°方向へ測って移動（距離が脇に表示・数値入力可）
  • 長押し（0.5秒）またはダブルクリック＝自由移動（機点スナップあり。押したままドラッグ→離して確定）
  • 右クリック＝向きを送る（Z軸45°×8→X軸45°×8）／Shift＋右クリック＝回転（旧ひねり・45°×8）
  • パイプ＝選択して端の点をクリックで起点選択 → 「長さ」入力かドラッグで伸縮（選んだ端が固定）
  • 仕様の変更＝配置済みの部品を選択すると左のパレットにその部品の仕様が表示され、パレットで呼び径・タイプ・クラス・Sch・長さなどを変えるとその部品が置いたまま作り替えられます（位置・向き・材質・グループは保持。「元に戻す」対応）
  • Esc＝取消／Del＝削除

まとめて選ぶ

  • Ctrl＋クリック＝複数選択（トグル）、Ctrl＋ドラッグ＝窓選択（囲んだものをまとめて選択）
  • 右のアイテムリストの行クリックでも同じ仕様をまとめて選択できます

リボン：ファイル

  • 新規＝図面をまっさらに（部品・寸法・図面情報をすべて消去。確認あり・直後なら「元に戻す」で復元できます）
  • 保存＝図面を .p3d.json で保存（部品・向き・材質・図面情報・注釈をまるごと）。2回目からは「上書き保存／新規保存」の選択メニューが出ます（PCのChrome/Edgeは同じファイルへ直接上書き。iPadは同名でダウンロードし「ファイル」で置き換え）
  • 開く＝保存した図面を読み込み（現在の内容は置き換わります）。開いたファイルがそのまま上書き保存の対象になります
  • CSV＝部品表をCSVで書き出し（Excelでそのまま開けます）。ガスケット・ボルト・溶接口・パイプ合計長さの自動集計付き（機点の接続から推定した参考値。アイテムリスト下部にも表示）
  • 自動保存＝作業内容を20秒ごとにブラウザ内へ自動バックアップ。誤って閉じても次回起動時に「復元しますか？」と出ます
  • 干渉（編集グループ）＝部品同士のぶつかりを検査して赤表示（接続済みの部品は除外・参考値。クリックで解除）
  • 画面を広く使う＝左上のアイテムパレット・右下のアイテムリスト・設計仕様・図面情報は、それぞれの見出しをクリック（タップ）で折りたたみできます（状態は記憶）。配置済みオブジェクトを選択しても空間上のEL入力フォームは出なくなりました（値の確認・編集はプロパティパネルで。移動コマンドの距離入力だけ従来どおり表示）
  • プロパティ（編集グループ）＝選択したオブジェクトの値を左下のパネルに一覧表示・編集できます。部品＝EL・パイプ長さ・フェイス面の方角（右/前/左/後＋方位角・立面角）・材質。線分/構築線＝起点EL＋終点は起点に対する相対（ΔX/ΔY/ΔZ）・長さ・水平角・立面角。円＝中心・半径。寸法＝実測値・値の上書き・逃げ・スライド角。文字＝内容・回転・配置点。ドラッグ中も値が追従表示されるので、空間での直接編集と併用できます（数値はEnterで確定・変更は「元に戻す」対応）。パネルは見出しをドラッグで好きな位置へ移動できます（位置は記憶・✕で閉じる）
  • 非表示／再表示（編集グループ）＝選択したアイテムを一時的に隠せます。①アイテムを選択→「非表示」②「非表示」→隠すアイテムをタップ、のどちらでも可（1回で終了・Esc・再押下で取消）。隠したアイテムは選択・スナップ・干渉チェックの対象外になりますが、アイテムリスト・自動集計・保存ファイルには残ります（リストでは淡色表示）。「再表示」ですべて元に戻ります（「元に戻す」でも戻せます）
  • 図面情報の既定値＝右下の図面情報の「この内容を既定として記憶」を押すと、設計仕様・場所・社名が記憶され、以後の新規図面に自動で入ります
  • 印刷＝図面（白地・線画）＋3D方位コンパス＋アイテムリスト・設計仕様・図面情報をA3横で印刷（アイテムリスト／設計仕様は折りたたみ中なら省略）／画像＝画面をPNG書き出し

リボン：描画

  • 線分＝起点で押す→ドラッグ→離して終点（離すと確定し、脇の入力欄で長さを微調整可）／構築線＝方向を決め両側へ無限／円＝中心をクリック→半径まで広げて確定（水平な円）／寸法＝2点を取り逃げを決めて寸法を入れる／文字＝点をクリックして配置→そのまま入力。配置後はクリックで選択（ドラッグで移動・Delで削除・右クリックで45°ずつ回転／長押しで角度スピナー）／ダブルクリックで文字を再編集。文字アイコンを右クリック＝色（シアン／白／黒／赤・既定シアン）と飾り（枠なし／枠あり／下線／二重下線）を選択。配置済みの文字を選択した状態でこのメニューを選ぶと、その文字の色・飾りも変わります
  • 円を選択すると中心＋四半円点（上下左右）が出ます。四半円点を掴んでドラッグ＝半径変更。Shiftを押しながら掴んで動かすと、その軸だけ伸びて楕円になります（通常は真円のまま拡縮）
  • 円も線分と同じ回転：選択して右クリック＝中心まわりに45°回転（Shift＝鉛直面で起こす）。右クリック長押し＝角度スピナーで任意角。楕円の長軸を傾けられます
  • 寸法アイコンを右クリック＝種別を選択：長さ(水平/垂直の距離・既定)／平行(2点間の距離)／角度／半径(R)／直径(φ)／引出(注記)。長さ・平行は2点で測り、長さは逃がした軸方向の成分だけを測ります（CADの長さ寸法）
  • 引出（リーダー）＝1点目＝矢印で指す点、2点目＝肘（位置）をクリック。肘から水平の棚が自動で伸び、確定と同時に入力フォームが開くのでそのまま文字を入力（既定値なし）。再選択すると入力フォームが開いて再編集できます
  • 角度＝2本の直線（線分・構築線）をそれぞれクリックして選択（選択中は緑にハイライト。直線以外をクリックしても反応しません）。最後にカーソルの象限で角度（θ／180−θ）・円弧半径を決めてクリックで確定（機点・端点・中点・交点にスナップ＝緑印）。補助線は引きますが対象の直線と重なる区間は表示しません。確定後は半径・直径と同じく、線/円弧クリックで選択→Del削除、値クリックで編集できます
  • 寸法の削除＝寸法線・円弧・補助線（＝値以外の部分）をクリックして選択しDel。値の編集＝赤い値をダブルクリックすると入力フォームが開き、数値・文字を上書きできます（Enter/Escまたは他をクリックで確定・閉じる。シングルクリック＝選択）。半径・直径・角度も同じ操作です。値をドラッグすると値だけを好きな位置へ動かせます（寸法線から離すと引出線を自動表示・元の位置近くへ戻すと解除。値が混み合う時の逃がしに）
  • 値の見やすさ（自動）＝値の背面には背景色の下地が敷かれ、線が数字を突き抜けません（印刷では白地）。値の大きさはズームアウトしても最小サイズを保ち、尺度表示（右上の尺度選択）・印刷では紙の上で常に約3.5mmになるよう自動調整されます。新しく入れた寸法の値が既存の値と重なる場合は自動で1段ずらして配置します
  • 寸法線の整列＝寸法線の本体をドラッグして逃げ（足の長さ）を調整中、他の寸法線の矢印と揃う位置で吸着します（相手の矢印が緑に光る）＝寸法線どうしを一直線に並べられます
  • 半径・直径は円／楕円をクリックすると寸法が出ます（近づけただけでは出ません）。その後カーソルを動かすと値の位置を内側／外側に調整でき、補助線（リーダー）が伸び縮みします（向きもカーソル方向に追従）。中心・四半円点・機点にスナップし、向きは四半円方向（上下左右）にも吸着します（緑印）。もう一度クリックで確定。Esc／右クリックでやり直し
  • 確定後の再調整：半径・直径・角度の寸法を選択（線/円弧をクリック）すると、再びカーソルで逃げ（リーダー長・内外／円弧の半径・向き）を調整できます（スナップ有・もう一度クリックで確定・Escで中断）
  • 起点・終点は部品の機点や他の線・寸法線の端点へスナップ（吸着すると緑の印が出て、そこへ引き寄せられます）
  • ファイル（リボン左端）＝クリックまたは長押しで新規・保存・開く・印刷・画像・CSVのメニューが開きます
  • 設定（リボン右）＝スナップ・表示のON/OFF。起点の表示＝選択中の起点・機点マーカー、交点の表示＝線どうしの交点の点表示と吸着。スナップ＝機点・端点・中点・交点への点吸着（OFFで完全フリー配置）。近接点＝ONの間、線分・構築線の線上にも吸着し、作図の2点目では相手の線への垂線の足（直角点）を優先して直角がぴったり出ます。いずれも状態は記憶
  • 斜めに引くと補助三角形が出て、X方向・Z方向の脚に長さ(mm)欄が表示。数値を入れてEnterで正確な寸法に確定（部品の方向移動と同じ操作）
  • Shiftを押しながら＝Y方向（鉛直）に引く（入力欄は「Y」）。水平角は45°刻み固定で拘束されます
  • 描画中：右クリック／Esc＝取消・解除、Del＝直近の注釈を取消／右ドラッグ＝移動・ホイール＝拡大
  • リボンの線分・円アイコンを右クリック＝線種を選ぶ（色は線種で固定：実線＝白／破線＝黒／点線＝青／一点鎖線＝赤）。線分・円・文字は、オブジェクトを選択した状態でリボンのアイコンを右クリックすると、その選択オブジェクトの書式（線種／色・飾り）を変更できます（未選択なら以後に描くものの既定設定）
  • 描き終えた線は（描画モードでない時）クリックで選択＝本体ドラッグで移動／端の青ハンドルをドラッグで長さ変更（脚・L欄で数値確定も可）。Delで削除、Escで選択解除

リボン：編集 ／ 画面右上のボタン

  • 戻す（Ctrl+Z）／やり直し（Ctrl+Y・Ctrl+Shift+Z）＝直前の操作を取り消し／再実行（配置・移動・削除・作図・編集など全般。最大80段）
  • 複製＝選択（部品・線分・構築線・寸法線）を100mm斜めにずらしてコピー／削除はDelキー
  • 鏡＝選択して押すと対話モード：①反転軸の起点をクリック（機点・交点に吸着・緑マーカーで候補表示）→②方向をクリック（カーソルが指す方向へ45°刻みで反転・青いゴーストのプレビューが追従）→手元の「オブジェクトを削除しますか？ はい／いいえ」で元を残すか選ぶ。線も反転対象。取消はEscか鏡ボタン再押下
  • グループ化＝複数（部品・線・寸法など）を選択して押すと一つのグループに。以後はどれか一つをクリックすると全部まとめて選択され、移動・削除も一緒に行えます。解除＝選択したグループを解除（保存ファイルにも引き継がれます）
  • 画面右上：家アイコン＝ホーム（最初の視点）／その下の範囲ズーム＝全体（選択があれば選択）が画面に収まるよう視点を移動／その下の尺度＝1:50・2:1・100%等を選ぶとその縮尺で表示（平行投影に切替・現在尺度を下に表示）。平行投影中はホイールで拡縮可
